# Widget Mond — конфигурации для виджета даты и времени на macOS

**Mond** — это набор конфигураций для программы **Übersicht** (`http://tracesof.net/uebersicht/`). Виджет отображает на рабочем столе macOS текущий день недели, дату и время в стильном минималистичном оформлении. Шрифт **Anurati** (прилагается) делает заголовок дня недели красивым декоративным шрифтом.

---

## Как установить Übersicht

Übersicht — это бесплатная программа для macOS, которая позволяет размещать виджеты на рабочем столе.

### Способ 1: Через Homebrew (рекомендуется, быстро)

Откройте **Терминал** и выполните:

```
brew install --cask ubersicht
```

Homebrew нужно предварительно установить, если его нет: https://brew.sh

### Способ 2: Скачать с официального сайта

1. Перейдите на сайт: http://tracesof.net/uebersicht/
2. Нажмите кнопку **Download**
3. Откройте скачанный `.dmg` файл
4. Перетащите **Übersicht** в папку **Программы (Applications)**
5. Запустите Übersicht из Launchpad или Programmes

После запуска Übersicht появится в строке меню (сверху, справа). Его иконка выглядит как квадратики.

---

## Как установить шрифт Anurati

В папке **widget-Mond** есть файл `Anurati.otf` — его нужно установить в macOS:

1. Откройте папку **widget-Mond**
2. Дважды кликните по файлу **Anurati.otf**
3. Нажмите **Установить шрифт** (в приложении **Шрифты** / Font Book)
4. Шрифт установлен

> **Если после этого шрифт не отображается**, потребуется ручная правка. Подробнее — в конце документа.

---

## Как установить виджет

1. Скопируйте папку **widget-Mond** в удобное место
2. Кликните по иконке Übersicht в строке меню (сверху справа)
3. Выберите **Open Widgets Folder** — откроется папка `~/Library/Application Support/Übersicht/widgets/`
4. Перетащите (скопируйте) всю папку **widget-Mond** в эту папку
5. Кликните по иконке Übersicht снова и выберите **Refresh**
6. В списке появятся все доступные конфигурации Mond
7. Нажмите **Enable** (или **Включить**) на нужной конфигурации (например, `mond-classic.jsx`)

Готово! Виджет появится на рабочем столе.

## Как выбрать конфигурацию (какой jsx-файл включать)

Все файлы разложены по папкам:

| Папка | Что внутри | Файлы |
|---|---|---|
| `originalMonds/` | Базовые версии | `mond-classic.jsx` — классический вид (день/дата/время) |
| | | `mond-classic-v2.jsx` — с декоративными символами |
| | | `mond-russian.jsx` — дни/месяцы **на русском** |
| | | `mond-all-Russian.jsx` — то же самое |
| `locale/` | Форматы даты | `mond-24h.jsx` — время в 24-часовом формате |
| | | `mond-europe.jsx` — дата в европейском формате (дд.мм.гггг) |
| | | `mond-iso.jsx` — дата в ISO-формате (гггг-мм-дд) |
| | | `mond-ordinal.jsx` — дата с окончаниями (1st, 2nd, 3rd) |
| `minimal/` | Минималистичные | `mond-simple.jsx` — без шрифта Anurati, просто Helvetica |
| | | `mond-small.jsx` — маленький виджет в правом верхнем углу |
| | | `mond-thin.jsx` — тонкий шрифт (font-weight: 100) |
| `style/` | Стили | `mond-reverse.jsx` — время сверху, день снизу |
| | | `mond-divider.jsx` — с разделительной линией |
| | | `mond-brackets.jsx` — дата в квадратных скобках |
| | | `mond-compact.jsx` — компактный, день и время в одной строке |
| | | `mond-lowercase.jsx` — строчные буквы (не ЗАГЛАВНЫЕ) |
| `partial/` | Частичная информация | `mond-no-date.jsx` — без даты (только день + время) |
| | | `mond-no-day.jsx` — без дня (только дата + время) |
| | | `mond-no-time.jsx` — без времени (только день + дата) |
| | | `mond-only-day.jsx` — только день недели |
| | | `mond-only-time.jsx` — только время |

| | | `mond-social-ru.jsx` — русский язык + иконки |

Чтобы включить/выключить конфигурацию — нажмите на иконку Übersicht → выберите **widget-Mond** → включите нужный файл.

## Как менять положение виджета на экране

Виджет позиционируется через CSS-свойства в файле `.jsx`. Для изменения расположения:

1. Откройте нужный `.jsx` файл в **TextEdit** (в режиме обычного текста) или в любом редакторе кода
2. Найдите в нём следующие строки (обычно в начале):

```css
top: 20%;
left: 50%;
transform: translate(-50%, -50%);
```

- **top** — расстояние от верхнего края экрана. Чем больше число — тем ниже виджет
- **left** — расстояние от левого края
- `transform: translate(-50%, -50%)` — центрирует виджет относительно указанной точки

**Примеры:**
- По центру экрана: `top: 50%; left: 50%; transform: translate(-50%, -50%);`
- В левом верхнем углу: `top: 5%; left: 5%; transform: none;`
- В правом верхнем углу: `top: 5%; left: auto; right: 5%; transform: none;` (как в `mond-small.jsx`)
- Внизу по центру: `top: 80%; left: 50%; transform: translate(-50%, -50%);`

После сохранения файла нажмите **Refresh** в меню Übersicht.

## Как менять цвет текста

Цвет текста задаётся свойством `color`. Найдите в `.jsx` файле:

```css
color: #f7f3e8;
```

По умолчанию это кремово-белый. Вы можете заменить этот код на любой другой:

- `#ffffff` — белый
- `#000000` — чёрный
- `#ff0000` — красный
- `#00ff00` — зелёный
- `#0088ff` — синий

Также можно добавить тень текста (параметр `text-shadow`). Например:

```css
text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.5);
```

После сохранения файла нажмите **Refresh** в меню Übersicht.

## Как управлять виджетом (основные действия)

- **Обновить все виджеты**: иконка Übersicht → **Refresh**
- **Включить/выключить**: иконка Übersicht → выберите виджет → **Enable** / **Disable**
- **Открыть папку виджетов**: иконка Übersicht → **Open Widgets Folder**
- **Скрыть/показать все виджеты**: иконка Übersicht → **Toggle Widgets**

## Возможные проблемы

### Шрифт Anurati не отображается (вместо букв квадратики или обычный шрифт)

В редких случаях встроенный в файл шрифт может не загрузиться. Тогда нужно сделать следующее:

1. Убедитесь, что вы **установили шрифт Anurati.otf** на компьютер (см. раздел выше)
2. Откройте `.jsx` файл в текстовом редакторе
3. Найдите и **удалите** блок `@font-face { ... }` целиком — от строки `@font-face {` до закрывающей `}`. В нём содержится длинная закодированная строка (base64) со шрифтом.
4. Оставьте только название шрифта в `font-family`:

```css
font-family: 'Anurati', sans-serif;
```

После удаления `@font-face` шрифт будет браться с вашей системы — так как вы уже установили `Anurati.otf`, всё будет работать.
